# POST-INSTALLATION COMMANDS
# ============================================================================
# After installing requirements.txt, run the following commands:
#
# 1. Install Playwright browsers (REQUIRED):
#    playwright install
#    playwright install-deps  # Install system dependencies (Linux only)
#
# 2. For face-recognition, you may need to install dlib dependencies:
#    Windows: Download dlib wheel from https://github.com/z-mahmud22/Dlib_Windows_Python3.x/releases
#    Linux: sudo apt-get install build-essential cmake
#           sudo apt-get install libopenblas-dev liblapack-dev
#
# 3. FFmpeg system installation (if not using ffmpeg-python):
#    Windows: Download from https://ffmpeg.org/download.html and add to PATH
#    Linux: sudo apt-get install ffmpeg
#    macOS: brew install ffmpeg
#
# 4. For audio processing, you may need:
#    Windows: Install Visual C++ Build Tools
#    Linux: sudo apt-get install libsndfile1
